Khelo – Soccer & Football Club is a clean, modern, creative and responsive WordPress Theme

== Description ==

Khelo – Soccer & Football Club is a clean, modern, creative and responsive WordPress Theme. It’s suitable for any sports as soccer, football, Soccer Club etc. Also anyone can use it for any personals, business, soccer news, football news etc. This theme includes 03 awesome homepage demos and more inner pages layout with individual club pages, player pages, gallery pages which are easy to edit and customize to fit your needs.
